# API Endpoints

The backend exposes endpoints for recommendation and retrieval tasks.

| Endpoint            | Method | Description                        |
|---------------------|--------|------------------------------------|
| `/recommend`        | POST   | Get API recommendations from query |
| `/feedback`         | POST   | Submit user feedback               |
| `/status`           | GET    | Health check/status                |

Payload shapes, authentication, and response formats are detailed in the development branch.
