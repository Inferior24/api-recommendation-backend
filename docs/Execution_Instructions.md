# Execution Instructions

**Backend execution is performed on the development branch.**

To run the backend, clone the repository and checkout the development branch:

```bash
git clone https://github.com/Inferior24/api-recommendation-backend.git
cd api-recommendation-backend
git checkout development
# Follow development branch instructions in its README for setup and execution
```

Backend setup involves:
- Installing dependencies (Python 3.8+, pip requirements)
- Preparing datasets and embeddings
- Starting the API server

All details are maintained in the development branch's `README.md`.
